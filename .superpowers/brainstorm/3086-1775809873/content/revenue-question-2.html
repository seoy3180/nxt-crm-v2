전년 비교 표시 방식

연간 현황 탭에서 "전년 비교" 버튼을 켰을 때 어떻게 보여줄까요?

A

겹침 차트 (Overlay)

같은 차트에 올해(진한색) + 작년(연한색) 막대를 겹쳐서 표시. 월별 직접 비교가 쉬움.

연한색 = 작년 / 진한색 = 올해

B

증감율 표시

올해 차트만 표시하되, 각 막대 위에 전년 대비 증감율(%) 라벨을 추가. 깔끔하고 정보 밀도 높음.

+12%
+25%
-8%
+5%
C

후순위로 미루기

현재 더미 데이터에 작년 데이터가 없고, 실 사용까지 시간이 있으니 전년 비교는 나중에 구현. 버튼만 disabled 상태로.
