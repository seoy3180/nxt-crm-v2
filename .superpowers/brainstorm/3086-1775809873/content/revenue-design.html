기술 설계 요약

데이터 소스
  • 연간 현황: contracts.total_amount + created_at 기준 월별 집계
  • 팀별 분석: contract_teams.percentage × total_amount 합산
  • 타입 필터: contracts.type (msp/tt/dev)
  • 미배분: contract_teams에 행이 없는 계약 금액
접근 권한
  • staff: /revenue 접근 차단 (사이드바에서 숨김)
  • team_lead: 소속 팀 매출만 표시
  • admin / c_level: 전사 매출 전체 표시
차트
  • Recharts BarChart (이미 설치됨)
  • 전년 비교: 겹침 막대 (올해 진한색 + 작년 연한색)
  • 타입별 색상: MSP=#2563eb, 교육=#f59e0b, 개발=#71717a
확인 필요 (진성님)
  • 매출 인식 기준 (생성일 vs 단계 vs 별도 필드)
  • MSP MRR 월별 분할 여부
  • 통합 등급 산정 기준
