### 4. 박진성 대표님이 "고객 담당자"로 등록된 케이스

노션의 "고객 담당자" 컬럼은 다른 계약건에서 사내 사람이 아닌 외부 인물이 적혀 있어서, 저는 **고객사 쪽 담당자**로 이해하고 CRM에 옮겼습니다. 그런데 아래 세 건은 "고객 담당자"에 **대표님(박진성)** 이 들어가 있는데, 그대로 유지해도 될지 확인 부탁드립니다.

| 고객사 | 고객 담당자 |
|---|---|
| 아주대학교_벤처창업 | 박진성 |
| 아주대학교_COSS | 박진성 |
| jspark | 박진성 |

## 정리 방침 (제안)

답변 주시면 아래처럼 정리하려고 합니다.

- 오기재 (`고려대학교 _DAMI연구실` 같은 경우) → **수정**
- `비고` → "미지정" 의미면 **값 없음**으로 변경
- 괄호 부가정보 → 의미에 따라 **분리 또는 유지**
